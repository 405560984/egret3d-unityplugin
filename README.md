# Egret3DExportTools
* 支持Unity5.6.4版本以上

## 使用方法
1. 复制Egret3DExportTools文件夹到Unity项目Assets下.
2. Unity编辑器菜单会多出一个Egret3DExportTools选项，点击打开即可.

## 注意事项
1. 导出NavMesh功能暂不支持.
